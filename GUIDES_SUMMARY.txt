================================================================================
                    QUICK REFERENCE
================================================================================

GET vs POST:
- GET: Data in URL (visible)
- POST: Data in body (hidden)

Validation:
- Always validate server-side
- Use filter_var() for emails
- Check length and type

XSS Prevention:
- htmlspecialchars() for output
- Input validation
- Content Security Policy

SQL Injection Prevention:
- Prepared statements
- Parameterized queries
- Input validation

CSRF Prevention:
- CSRF tokens
- SameSite cookies
- Origin header check

File Upload Security:
- Whitelist extensions
- Check MIME type
- Validate content
- Random filename
