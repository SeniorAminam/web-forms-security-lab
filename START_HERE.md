## 💡 نکات مهم

### ✅ بهترین روش‌ها

1. **به ترتیب پیش بروید**
   - هر مرحله بر پایه قبلی است

2. **مثال‌ها را تست کنید**
   - فقط خواندن کافی نیست

3. **خطاها را ایجاد کنید**
   - تغییرات کنید و ببینید

4. **نکات امنیتی را یادبگیرید**
   - چرا و چطور

### ⚠️ هشدارها

- ⚠️ فقط برای آموزش است
- ⚠️ هرگز در تولید استفاده نکنید
- ⚠️ همیشه validation انجام دهید
- ⚠️ HTTPS را استفاده کنید

## 🔍 مثال: اولین دمو

### کاری که باید کنید:

**1. سرور را اجرا کنید:**
```bash
php -S localhost:8000
```

**2. Dashboard را باز کنید:**
```
http://localhost:8000/dashboard.php
```

**3. روی "GET vs POST" کلیک کنید:**
```
01_get_post.php باز می‌شود
```

**4. فرم GET را تست کنید:**
- "لپ‌تاپ" را بنویسید
- ارسال کنید
- URL تغییر می‌کند: `?query=لپ‌تاپ`

**5. فرم POST را تست کنید:**
- "رمز" را بنویسید
- ارسال کنید
- URL تغییر نمی‌کند

**6. راهنما را بخوانید:**
👉 [`GUIDE_01_GET_POST.md`](GUIDE_01_GET_POST.md)
